# SIBA — Sherazi IT Business Academy

![Banner](/public/logo.svg)

## 🚀 Overview

**SIBA (Sherazi IT Business Academy)** is a state-of-the-art, high-fidelity Training and Business Modernization platform. It moves beyond passive video watching by integrating **AI-guided adaptive learning paths**, hands-on **business execution tracking**, and a structured **6-step mastery propulsion system**.

Built with a premium 3D glassmorphism aesthetic, SIBA provides a seamless powerhouse experience for Students, Trainers, Mentors, and Administrators to learn, execute, and scale.

---

## ✨ Key Features

### 🧠 AI-Powered Modernization Engine
*   **Adaptive Paths**: The system detects student weaknesses and adjusts course material pacing dynamically.
*   **Skill Validation**: Automatic grading and feedback for practical assignments using AI evaluation models.
*   **Guided Execution**: Real-time business coaching for building scalable systems and modernizing IT infrastructure.

### 📊 Elite Multi-Role Dashboards
*   **Student Hub**: Focus on execution with business KPI tracking, project hubs, and automated certification vaults.
*   **Trainer Portal**: Command center for course OS design, bulk student management, and performance analytics.
*   **Mentor Console**: Dedicated workspace for cohort guidance, scheduling, and targeted feedback loops.
*   **Admin Command Center**: Full platform governance, revenue operations (MRR/ARR), and system-wide settings.

### 🎨 State-of-the-Art Design
*   **Glassmorphism Ecosystem**: Crystal-clear surfaces, dynamic blurring, and premium saturated gradients.
*   **Propulsion Animations**: Smooth slide-in-ups, floating orbs, and responsive micro-interactions.
*   **Mobile-First Precision**: Tight, seamless layout optimized for high-performance mobile scrolling.

## 🛤️ The Mastery Propulsion System

SIBA enforces a strict, scientifically-backed 6-step propulsion sequence to ensure actual revenue-generating mastery:

1.  **Orientation**: Mindset recalibration and strategic direction setting.
2.  **Core Learning**: Deep-dive into elite theoretical concepts.
3.  **Practical**: Mission-critical execution with real-world scenarios.
4.  **Evaluation**: Rigorous AI-verified testing of skills.
5.  **Certification**: Issuance of verifiable, expert-grade credentials.
6.  **Business**: Launching and scaling the skills into a functional Business Operation.

## 🚦 Getting Started

### Prerequisites
*   Node.js 20+
*   NPM or PNPM

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/beingmushfiq/atcms.git
    cd atcms
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Environment Setup**
    Create a `.env` file in the root:
    ```env
    DATABASE_URL="your_database_url"
    NEXTAUTH_SECRET="your_secret"
    ```

4.  **Database Migration & Seeding**
    ```bash
    npx prisma generate
    npx prisma db push
    npm run seed
    ```

5.  **Run Development Server**
    ```bash
    npm run dev
    ```
